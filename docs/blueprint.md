# **App Name**: 바이브챗

## Core Features:

- 사용자 계정 관리: 사용자는 Supabase Auth를 사용하여 안전하게 회원가입, 로그인 및 로그아웃할 수 있습니다.
- 1:1 메시징: Supabase Realtime을 통해 다른 등록된 사용자와 실시간 비공개 텍스트 메시지를 주고받을 수 있습니다.
- 대화 보기: 선택한 1:1 대화 내의 메시지 기록을 시간순으로 표시합니다.
- 사용자 검색 및 선택: 사용자 이름 또는 ID를 기반으로 다른 등록된 사용자를 검색하여 새 대화를 시작할 수 있습니다.
- AI 답장 제안: AI 도구를 활용하여 상황에 맞는 짧은 답장 제안을 받아 더 빠른 의사소통을 돕습니다.

## Style Guidelines:

- 평온한 하늘의 차분한 파란색 #3377CC를 주 색상으로 사용하여 신뢰감과 연결감을 불러일으킵니다. 이 색조는 밝은 배경에 대해서도 눈에 띄도록 충분히 깊습니다.
- 매우 밝고 채도가 낮은 파란색 #EFF3F8이 배경색으로 사용됩니다. 거의 흰색에 가까운 이 색상은 가독성과 개방감을 보장하며, 주 색상과 경쟁하지 않고 보완합니다.
- 생생한 시안색 #4DCDE5가 강조 색상으로 사용됩니다. 이 색상은 주 색상의 유사 색상에서 파생되었지만, 상호 작용 요소에 시선을 끌고 새로운 정보를 강조하기 위해 밝고 채도가 높습니다.
- 모든 텍스트는 'Inter' (산세리프) 폰트를 사용합니다. 이 폰트는 메시징 애플리케이션에서 흔히 나타나는 장시간 읽기에 이상적인 뛰어난 가독성과 중립적이면서도 현대적인 외관을 위해 선택되었습니다.
- 깔끔하고 현대적인 인터페이스를 유지하기 위해 필수적인 액션(예: 전송, 검색, 프로필)에는 현대적이고 선명한 라인 아트 벡터 아이콘이 사용됩니다. 아이콘은 단색이거나 강조 색상과 일치하는 미묘한 그라데이션을 사용해야 합니다.
- 레이아웃은 넓은 화면에서는 고전적인 두 개의 창 디자인(왼쪽에 대화 목록, 오른쪽에 채팅 창)을 특징으로 하며, 작은 화면에서는 최적의 반응성을 위해 단일 창의 탭 방식으로 전환됩니다. 메시지는 깔끔하고 비대칭적인 채팅 버블로 표시됩니다.
- 메시지 전송 및 수신(예: 짧은 페이드인 또는 슬라이드업), 새 메시지 알림, 그리고 채팅 창 간의 부드러운 전환을 위해 미묘한 애니메이션이 통합되어 사용자 경험을 방해하지 않으면서 향상시킵니다.
